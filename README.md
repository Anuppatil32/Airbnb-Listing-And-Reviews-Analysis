🏡 Global Airbnb Performance Dashboard – Power BI Project

📌 Project Overview

✔This project is an interactive Power BI Dashboard built to analyze and visualize global Airbnb listings and reviews data across multiple cities.
The dashboard provides insights into:

🔹Total listings and reviews
🔹Market share by city
🔹Property type performance
🔹Average pricing analysis
🔹Rating comparison
🔹Airbnb growth lifecycle over time

✔The aim of this project is to help users understand Airbnb business trends, customer preferences, and city-wise market performance using data visualization techniques.

📊 Dashboard Highlights

🔹 KPI Metrics
279,712 Listings
182,024 Hosts
17,589K Reviews
144 Property Types
10 Major Cities Analyzed

🔹 Key Insights
Paris, New York, and Sydney contribute nearly half of the total listings and reviews.
Paris has the highest number of listings and reviews among all cities.
Hotel rooms are priced almost 2x higher than Airbnb listings in some locations.
Mexico City recorded the highest average rating.
Airbnb experienced major growth between 2013–2016 before slowing down due to regulations and COVID-19 impacts.
